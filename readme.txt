Corpo de script para rodar testes automatizados usando selenium:

Cada navegador possui o seu proprio driver, portanto eles devem estar localmente instalados na maquina e configurados através do PATH.

Funcionamento: Jest é uasdo para gerar o relatorio e fazer as comparações dos testes e o selenium para interagir com os navegadores, Chrome e IE11.

Esse projeto sera usado como conceito para outros futuros.
